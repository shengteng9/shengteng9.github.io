什么是帧动画？

所谓帧动画就是在“连续的关键帧”中分解动画动作，也就是在时间轴的没帧上逐帧绘制不同的内容，使其连续播放而成动画由于是一帧一帧的画。所以帧动画具有非常大的灵活性，几乎可以表现任何想表现的内容。

常见的帧动画方式

GIF
CSS3 animation
javascript

GIF和CSS3 animation实现帧动画存在着不足

1.不能灵活的控制动画的暂停和播放
2.不能捕捉到动画完成的事件
3.不能对帧动画做更加灵活的扩展

js中有三种继承方式

js原型（prototype）实现继承

function Person(name,age){  
    this.name=name;  
    this.age=age;  
}  
Person.prototype.sayHello=function(){  
    alert("使用原型得到Name："+this.name);  
}  
var per=new Person("rudy",21);  
per.sayHello(); //输出：使用原型得到Name:rudy`
