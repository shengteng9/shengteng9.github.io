CSS单位 px,em,rem

  • px:1个px相当于一个像素,精确定位
  • em:相对的长度单位
    1. em相对参照物为父元素的font-size
    2. em具有继承的特点
    3. 当没有设置font-size时，浏览器会有一个默认的em设置：1em=16px
  • rem:相对的长度单位（参照物为根元素font-size）

如何实现响应式图片

js或服务端

通过监听屏幕大小改变图片路径，实现加载不同分辨率图片：

$(document).ready(function(){
    function makeImageResponsive(){
        var width=$(window).width();
        //.xxx 图片父容器
        var img=$('.xxx img');
        if(width <= 480){
            img.attr('src','img/480.png);
        } else if(width <= 800){
            img.attr('src','img/800.png)
        } else{
            img.attr('src','img/1600.png)
        }
    }
    $(window).on('resize load',makeImageResponsive);
});
或将屏幕信息写入cookie，在获取图片时返回不同图片。
srcset
<img class="image" src="img/480.png srcset="img/480.png 480w,img/800.png 800w,img/1600.png 1600w">

当图片父容器设置成50%时，浏览器亦然按照100%的临界去加载

srcset配合sizes

解决只使用srcset出现的问题
默认值sizes=”100vw” （100viewport）代表100%的适口宽度

<img class="image" src="img/480.png 
srcset="img/480.png 480w,img/800.png 800w,img/1600.png 1600w"
sizes="(min-width:800px,calc(100vw - X em),100vw)"
>

sizes=”(min-width:800px,calc(100vw - X em),100vw)”:
适口宽度超过800的时候图片预估宽度即最大为calc(100vw - X em)宽，小于则是100适口宽度

picture标签实现多种条件匹配
<picture>
    <source media="(max-width:36em)"
    srcset="img/xxx-x.jpg  768w"/>
    <source
       srcset="img/xxx.jpg 1800w />
       //设置图片格式
    <source 
       type="image/webp" srcset="logo.webp 480w,logo-m.webp 800w,logo-l.webp 1600w />
    <img class="image" src="img/xxx.jpg">
</picture>
