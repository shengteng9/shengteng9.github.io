Typescript是什么

TypeScript 是 Microsoft 开发和维护的一种面向对象的编程语言。它是 JavaScript 的超集，包含了 JavaScript 的所有元素，可以载入JavaScript 代码运行，并扩展了 JavaScript 的语法。

为什么用Typescript

  • TypeScript 增加了静态类型、类、模块、接口和类型注解。（关于静态类型系统看这篇文章）
  • TypeScript 可用于开发大型的应用。
