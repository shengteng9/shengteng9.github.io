Node clinic

Node clinic是一款诊断Node.js性能监控的可视化工具，有较丰富的功能。主要包括Doctor，Flame，Bubbleprof三部分。我只对Doctor做了尝试。
Doctor可以帮助您诊断应用程序中的性能问题，并指导您使用更专业的工具来深入了解您的具体问题。诸如CPU使用率低，阻止垃圾收集，频繁的事件循环延迟或混乱数量的活动句柄等症状可能表明存在许多潜在问题。 Doctor通过根据这些症状生成推荐来帮助缩小可能性。 诸如I / O问题，非优化垃圾收集和阻塞事件循环等示例非常常见。 Doctor会帮助你解决所有这些问题。
